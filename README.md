# Обработка транзакций

## Описание

"Обработка транзакций" - это Python-приложение для фильтрации и сортировки финансовых транзакций.

## Установка

1. Клонируйте репозиторий:
  ```
  git clone https://github.com/darialastochkina/homework.git
  ```
2. Установите зависимости:
  ```
  pip install -r requirements.txt
  ```

## Использование

1. Импортируйте функции в ваш Python-код.
2. Используйте `filter_by_state()` для фильтрации транзакций по статусу.
3. Используйте `sort_by_date()` для сортировки транзакций по дате.
4. Используйте `filter_by_currency()` для фильтрации транзакций по валюте.
5. Используйте `transaction_descriptions()` для получения описаний транзакций.
6. Используйте `card_number_generator()` для получения номеров карт.
7. Используйте `@log(filename=...)` для логирования выполнения результата функции и ее параметров
